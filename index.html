The Basics of the Web and HTML


The World Wide Web
It's a collection of HTML documents. The links between the pages are called Hyper Links.

File Types
On the web we can find plain text, HTML, images, videos and music.

Components of the Web
Browser: A program that runs on your computer to display files found on the web
The internet: The world's largest computer network
HTTP: The main protocol of the web
Servers: Computers that host the files that make up the web

HTML Basics
HyperText Markup Language is made by text content, markup, references to other documents and links to other pages.

HTML Tags
Tags are structured with an opening and closing tag, which have in the middle the content. The whole thing is refered to as the element.

Why computers are studpid
 It is very important that we are carefull and we code correctly. Otherwise, the computer will not understand your order and mistakes will start to appear, as it will follow your orders literally.

Inline vs Block elements
 HTML elements can be inline or block. Elements like anchor (a) are inlines whilst others like paragraph (p) are blocks. Block makes an invisible box around the element, that has a height and width whilst inline elements are just text.
